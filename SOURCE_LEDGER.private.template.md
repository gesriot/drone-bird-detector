# Private Source Ledger Template

Do not commit a filled copy of this file. Copy it to
`SOURCE_LEDGER.private.md` and keep it outside public Git history.

## Dataset Entry Template

```text
Code:
Private/source name:
Source URL:
Download date:
Version/export id:
License:
License URL or local copy:
Citation:
Original class map:
Public class remap:
Split rule:
Known restrictions:
Checksums:
Notes:
```

## Why This Exists

The public repository uses anonymized dataset codes (`dataset01`, `dataset02`,
and so on) because source data is not redistributed. This private ledger is the
place to keep enough detail to reproduce the work and satisfy attribution or
audit requirements.
